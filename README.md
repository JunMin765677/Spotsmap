# SposMap

SpotsMap 是一個使用 Node.js + Express + MySQL + EJS 建立的電子商務全端專案，部署於 AWS Elastic Beanstalk，以 RESTFul API 滿足網站不同資料的互動需求，搭配 Docker、AWS CLI(EB CLI) 自動化部署，打造一個多功能的電商網站。
